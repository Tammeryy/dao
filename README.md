# Dao
Dao is a JavaScript library for graph traversal.

## Getting Dao
```
npm install --save dao
```

## Using Dao
With ES6 imports:
```
import dao from 'dao'
```

With CommonJS
```
var dao = require('dao')
dao.
```

As an SFX bundle:
```
<script src="dist/dao.js"></script>
```
